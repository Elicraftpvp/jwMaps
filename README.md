# JW Maps Forquilhinha

Projeto desenvolvido para auxiliar na organização e gerenciamento dos **mapas da congregação de Forquilhinha**, com foco em facilitar o trabalho do **servo de mapas**.

## Objetivo
O sistema tem como propósito centralizar e simplificar a gestão dos mapas, dirigentes e registros relacionados. Assim, é possível ter um melhor controle e acesso rápido às informações necessárias para o serviço.

## Estrutura do Projeto
- **login.html** – Tela inicial de autenticação.
- **site/**
  - **index.html** – Página principal após o login.
  - **backend/** – APIs e lógica em PHP para autenticação, dashboards, dirigentes e mapas.
  - **images/** – Recursos visuais utilizados no sistema.
  - **pages/** – Páginas específicas como dashboard, dirigentes e gerenciar mapas.
    - **views/** – Visualizações detalhadas, como mapas por dirigente.
  - **script/** – Arquivos JavaScript responsáveis pela interação e funcionalidades dinâmicas.
  - **style/** – Estilos do sistema em CSS.

## Funcionalidades
- **Login seguro** para acesso restrito.
- **Painel de controle (Dashboard)** com visão geral do sistema.
- **Gestão de dirigentes**, com cadastro e visualização.
- **Gerenciamento de mapas**, incluindo organização e atribuição.
- **Visualização de mapas por dirigente**, facilitando acompanhamento.
- **APIs em PHP** para integração entre frontend e backend.

---

> Este projeto foi criado com o intuito de apoiar o trabalho do **servo de mapas**, tornando a organização mais prática e eficiente.
